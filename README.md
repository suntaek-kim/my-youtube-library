# 나의 유튜브 서재

내가 사랑하는 유튜브 영상들을 한 권의 책처럼 모아두는 개인 큐레이션 사이트.
누구나 둘러볼 수 있고, 회원가입하면 자기만의 책장을 가질 수 있어요.

## 🌐 라이브 데모
[https://suntaek-kim.github.io/my-youtube-library/](https://suntaek-kim.github.io/my-youtube-library/)

## ✨ 주요 기능
- 📚 카테고리별 영상 카드 그리드 (책장 무드)
- 🔍 실시간 검색 — 제목·채널·코멘트·카테고리 통합 (200ms 디바운스)
- 🔐 회원가입 / 로그인 (Supabase Auth)
- ☁️ 클라우드 저장 — 어느 기기에서 접속해도 같은 책장 (Supabase DB)
- ⚡ 실시간 동기화 — 다른 기기에서 바꾸면 새로고침 없이 화면 갱신 (Supabase Realtime)
- 💾 JSON 백업 / 복원
- 🎨 Wanted Design System 기반 (Pretendard Variable + #0066FF)
- 📱 반응형 디자인 (모바일 1열 → 태블릿 2열 → 데스크톱 3열)

## 🚀 사용 방법

### 둘러보기 (비로그인)
URL에 그냥 접속하면 사이트 주인장의 책장을 구경할 수 있어요. 영상 보기·검색·재생이 가능해요.

### 회원가입 / 로그인
헤더 우측의 **"로그인 / 회원가입"** 버튼을 눌러 이메일과 비밀번호로 가입하세요.

### 본인 책장 관리 (로그인 후)
- ➕ 영상 추가 (우하단 + 버튼)
- ✏️ 편집 / 🗑️ 삭제
- 📤 JSON 내보내기 / 📥 가져오기
- 모든 변경은 클라우드에 저장되고, 다른 기기에서도 실시간으로 반영돼요

## 🛠️ 기술 스택
- **프론트엔드**: HTML, CSS, JavaScript (프레임워크 없는 바닐라 구성)
- **백엔드**: Supabase
  - **Auth** — 회원가입 / 로그인
  - **Database** (PostgreSQL) — 영상 데이터 저장
  - **Realtime** — 실시간 동기화
- **호스팅**: GitHub Pages

## 🔒 보안 모델
- 코드에 포함된 Supabase **anon key는 공개용 키**입니다. 공개돼도 안전하도록 설계된 키예요.
- 실제 데이터 보호는 **Supabase RLS(Row Level Security) 정책**이 강제합니다:
  - **읽기**: 누구나 모든 영상을 볼 수 있음 (공개 책장)
  - **추가 / 수정 / 삭제**: 로그인한 사용자가 **자기 영상에 대해서만** 가능
- `service_role` 키는 코드에 포함되지 않습니다.

## 💭 회고
HTML 파일 하나로 시작한 프로토타입이, localStorage를 거쳐 진짜 클라우드 서비스가 됐다.

단일 HTML 분리 → Git/GitHub → 디자인 시스템 → 실시간 검색 → 인증 →
클라우드 데이터베이스 → 실시간 동기화. 한 단계씩 쌓아 올린 결과,
집에서 추가한 영상이 회사 화면에 새로고침 없이 떠오르는 사이트가 완성됐다.
