## 🔐 관리자 모드
영상을 추가/편집/삭제하려면 관리자 모드로 진입하세요.

진입 방법 (둘 중 하나):
- URL 끝에 `?admin=true` 붙이기
- 페이지에서 `Ctrl + Shift + A` 단축키

비밀번호 입력 후 활성화되며, 브라우저 탭을 닫으면 자동 로그아웃됩니다.

## 💾 데이터 백업

영상 데이터는 브라우저 localStorage에 저장돼요. 다른 컴퓨터(예: 집 → 회사)로 옮기려면 백업이 필요합니다.

### 다른 컴퓨터로 옮기는 방법
1. 원본 컴퓨터에서 관리자 모드 진입 → "📤 내보내기" 클릭 → JSON 파일 자동 다운로드
2. 새 컴퓨터에서 관리자 모드 진입 → "📥 가져오기" → 다운로드한 JSON 파일 선택
3. 옵션 선택:
   - **병합**: 기존 데이터에 추가 (videoId 기준 중복 자동 제거)
   - **덮어쓰기**: 기존 데이터를 모두 삭제하고 교체

### 정기 백업 권장
영상이 10개 이상 쌓이면 내보내기 버튼 옆에 작은 점이 표시돼요. 그때 한 번 백업하시면 좋아요.
