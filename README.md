# 나의 유튜브 서재

내가 사랑하는 유튜브 영상들을 한 권의 책처럼 모아두는 개인 큐레이션 사이트.

## 🌐 라이브 데모
[https://suntaek-kim.github.io/my-youtube-library/](https://suntaek-kim.github.io/my-youtube-library/)

## ✨ 주요 기능
- 📚 카테고리별 영상 카드 그리드 (책장 무드)
- 🔍 **실시간 검색** — 제목·채널·코멘트·카테고리 통합 (200ms 디바운스, 카테고리 필터와 AND 조건)
- 🔐 **관리자 모드** — 방문자/관리자 분리, 비밀번호 게이트, sessionStorage 기반
- 💾 **JSON 백업/복원** — 집-회사 동기화용, 병합/덮어쓰기 옵션
- 🎨 Wanted Design System 기반 (Pretendard Variable + #0066FF Primary)
- 📱 반응형 디자인 (모바일 1열 → 태블릿 2열 → 데스크톱 3열)

## 🚀 사용 방법

### 방문자 (기본)
URL에 그냥 접속하면 영상 보기·검색·재생만 가능한 깔끔한 갤러리 모드예요.

### 관리자 (영상 추가/편집/삭제)
영상을 관리하려면 관리자 모드로 진입하세요. 두 가지 방법:
- URL 끝에 `?admin=true` 붙이기
- 또는 페이지에서 `Ctrl + Shift + A` 단축키

비밀번호 입력 후 진입. 탭을 닫으면 자동 로그아웃돼요.

### 백업 (집 → 회사 동기화)
- 관리자 모드 → "📤 내보내기" → JSON 파일 자동 다운로드
- 파일명 형식: `youtube-library-backup-YYYY-MM-DD.json`

### 복원 (다른 컴퓨터에서 불러오기)
- 관리자 모드 → "📥 가져오기" → JSON 파일 선택
- 옵션 선택:
  - **병합**: 기존 데이터에 추가 (videoId 기준 중복 자동 제거)
  - **덮어쓰기**: 기존 데이터를 모두 삭제하고 교체

> 💡 영상이 10개 이상 쌓이면 내보내기 버튼 옆에 작은 점이 표시돼요. 백업 권장 신호!
